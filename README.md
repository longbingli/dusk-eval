## 📌 项目简介

**Dusk-Eval** 是一款基于大语言模型（ChatGLM）构建的智能答题与自动评分系统，支持 AI 自动出题、实时流式输出（SSE）、智能评分与策略扩展能力。

系统采用 **Spring Boot + Redis + RxJava + SSE + ChatGLM** 构建，结合策略模式与注解驱动设计，实现高扩展、低耦合、可插拔式架构。

---

## ✨ 项目亮点

- 🤖 AI 自动出题（ChatGLM）
- ⚡ SSE 流式响应，边生成边返回
- 🧠 多策略评分体系（AI / 规则）
- 🧩 注解驱动策略扫描，替代 if-else
- 🔗 RxJava 处理 AI 流式数据
- 🔒 雪花算法 + 幂等设计
- 📚 Swagger + Knife4j 接口文档
